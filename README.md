Tres en Raya (versión mejorada)

Este proyecto consiste en una versión mejorada del juego anterior que tenia 3en raya.

Sistema de login de usuarios
Asociación de partidas y resultados a cada usuario
Sistema de récords
Se registran las victorias de cada jugador
Se mantiene un conteo acumulado de partidas ganadas
Conexión con backend:
Uso de fetch para enviar y obtener datos
Comunicación entre frontend y servidor
Los datos del juego se almacenan 
















----Funcionalidades base-----
Tablero 3x3 y 5x5
Turnos alternos entre el jugador y el ordenador 
Para automáticamente al ganar o quedar  empate
Reinicio de partida
----Mejoras añadidas----
-Sistema de récords
-Conexión básica con backend
-Uso de fetch para enviar y obtener datos
-Permite guardar y recuperar información de las partidas
-Se registran las victorias de cada jugador
-Los datos del juego se guardan para no perderse al recargar la página
 -Visualización de resultados
Se pueden consultar las victorias de cada jugador
